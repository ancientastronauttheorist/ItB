# Shop Grid-Power Priority

Grid Power purchases are mandatory only while the grid is below its maximum.
The purpose of the rule is to restore the grid to 7/7 before buying optional
gear, because grid power is the run-ending resource.

When the grid is already full, the shop's Grid Power slot becomes Overpower /
+Grid Defense. Treat that as an optional purchase, not an automatic spend. Buy
it when an achievement target needs full-grid power gains or Grid Defense, or
when there is no better use for reputation. Otherwise prefer reactor cores,
solver-supported weapons/passives, useful pilots, and other achievement-relevant
purchases.
